ISSUE: Build Failed
SOLUTION:
1. Check build logs in Vercel dashboard
2. Make sure root directory is set to "backend"
3. Verify package.json exists in backend folder
4. Check for syntax errors in code

---

ISSUE: "Route not found" error
SOLUTION:
1. Make sure you're accessing /api/health not just /health
2. All routes must start with /api/
3. Check vercel.json is in backend folder

---

ISSUE: Firebase error in logs
SOLUTION:
1. Verify FIREBASE_SERVICE_ACCOUNT_BASE64 is set correctly
2. Make sure there are no line breaks in the base64 string
3. Check the base64 string is complete (very long)

---

ISSUE: CORS error from frontend
SOLUTION:
1. Update FRONTEND_URL in Vercel environment variables
2. Set to your actual frontend URL (Netlify or localhost)
3. Redeploy backend after updating
