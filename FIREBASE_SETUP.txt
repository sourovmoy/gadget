ğŸ” TROUBLESHOOTING

Problem: "Authentication failed" error
Solution: 
- Make sure Email/Password is enabled in Firebase Console
- Check browser console for specific error
- Verify Firebase config in frontend/.env.local

Problem: "Invalid email" error
Solution:
- Use a valid email format (e.g., user@example.com)
- Email doesn't need to be real, just valid format

Problem: "Weak password" error
Solution:
- Password must be at least 6 characters
- Use: test123456 or any password with 6+ characters

Problem: "Email already in use" error
Solution:
- This email is already registered
- Try logging in instead of signing up
- Or use a different email
